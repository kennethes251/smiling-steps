## Most Common Fixes

### 1. **Refresh Everything**
   - Refresh browser page
   - Restart browser completely
   - Restart computer if needed

### 2. **Check Permissions**
   - Camera icon in address bar ‚Üí Allow
   - Microphone icon in address bar ‚Üí Allow
   - Refresh page after changing

### 3. **Clear Browser Data**
   - Press Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
   - Select "Cached images and files"
   - Clear and try again

### 4. **Switch Browsers**
   - Download Chrome if you don't have it
   - Try incognito/private mode first
   - Log in and test video call

### 5. **Check Network**
   - Move closer to WiFi router
   - Use ethernet cable if possible
   - Close Netflix, YouTube, downloads
