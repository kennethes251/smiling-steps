## Quick Start Workflow

### For Blog Posts (800-1500 words):
1. Choose category (Mental Health, Therapy Tips, Self-Care, etc.)
2. Replace [placeholders] with your content
3. Add 1-2 images
4. Review and publish

**Time estimate:** 30-45 minutes

### For Recovery Guides (2000+ words):
1. Choose "Recovery Guide" template
2. Work through each part systematically
3. Add multiple images and resources
4. Save as draft, review, then publish

**Time estimate:** 2-3 hours

### For Community Education (1500 words):
1. Choose "Community Education" template
2. Add local statistics and resources
3. Include FAQs relevant to your community
4. Add educational resources

**Time estimate:** 1-2 hours

### For Support Tools (1200 words):
1. Choose "Support Tool" template
2. Detail step-by-step instructions
3. Add practical examples
4. Include troubleshooting section

**Time estimate:** 1-1.5 hours

When using templates, remember to:

- [ ] Replace all [placeholder] text
- [ ] Add compelling title (under 60 characters)
- [ ] Write engaging excerpt (under 500 characters)
- [ ] Fill meta title (under 60 characters)
- [ ] Fill meta description (under 160 characters)
- [ ] Add 3-5 relevant tags
- [ ] Upload featured image
- [ ] Include internal links

## Common Questions

**Q: Can I edit a template after selecting it?**
A: Yes! Templates are just starting points. Edit freely.

**Q: Can I switch templates after starting?**
A: Yes, click "Change Template" button at the top.

**Q: What if I don't want a template?**
A: Click "Start from Scratch" at the bottom of template selection.

**Q: Can I save my own custom templates?**
A: Not yet, but this feature is coming soon!

**Q: Do I have to use all sections in a template?**
A: No, remove any sections that don't fit your content.
