## How to Use Templates

### Option 1: Use a Template
1. Click on any category card
2. The form will auto-fill with the template content
3. Replace placeholder text (in [brackets]) with your actual content
4. Customize as needed
5. Add your featured image
6. Save as draft or publish

### Option 2: Start from Scratch
1. Click **"Start from Scratch (No Template)"** at the bottom
2. Create content without any pre-filled text
3. Select category manually from dropdown

### Option 3: Change Template
- If you started with a template but want to switch:
- Click **"Change Template"** button at the top
- Select a different category template

## Best Practices

### When Creating Content:

1. **Replace All Placeholders**
   - Look for text in [brackets]
   - Replace with specific, relevant content

2. **Customize the Structure**
   - Add or remove sections as needed
   - Adjust headings to fit your topic

3. **Add Rich Content**
   - Include real examples
   - Add statistics and data
   - Use bullet points for readability

4. **Include Visuals**
   - Upload a featured image
   - Consider adding image URLs in content

5. **Optimize for SEO**
   - Fill in meta title and description
   - Use relevant tags
   - Write compelling excerpts

6. **Review Before Publishing**
   - Save as draft first
   - Preview the content
   - Check for completeness
   - Publish when ready

## Tips for Each Content Type

### For Blogs:
- Keep it conversational and engaging
- Use personal stories or examples
- Include actionable takeaways
- Aim for 800-1500 words

### For Recovery Guides:
- Be comprehensive and detailed
- Include step-by-step instructions
- Add downloadable resources
- Aim for 2000+ words

### For Community Education:
- Use clear, accessible language
- Include statistics and facts
- Provide multiple resource types
- Address common questions

### For Support Tools:
- Focus on practical application
- Include specific examples
- Add troubleshooting section
- Make it actionable
